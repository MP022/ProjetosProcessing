# ProjetosProcessing
Meus projetos feitos usando o ambiente de desenvolvimento do [processing.org](https://processing.org/)! Esses projetos foram feitos para explorar a minha criatividade e treinar as minhas habilidades com programação voltada para o visual e o desenvolvimento de jogos.

## Execução dos Projetos

Por causa das diferentes linguagens de programção esses dois tipos de exigem ambientes diferentes para serem executados.

 * **Processing**  
 Para executar esses projetos basta baixar a [IDE do processing](https://processing.org/download) e ao executar ela abrir o projeto que deseja executar.

 * **p5.js**  
 Para executar esses projetos pelo VS Code é preciso seguir o [tutorial](https://p5js.org/tutorials/setting-up-your-environment/#vscode) de como preparar o ambiente de desenvolvimento. Também existe a possibilidade de usar o editor web de **p5.js** e para isso é só seguir o [tutorial](https://p5js.org/tutorials/setting-up-your-environment/#web-editor).
